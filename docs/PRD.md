# PRD — 대진밥핏

> 대진대학교 학식 영양·열량 분석 웹앱의 제품 요구사항 문서입니다.

## 1. 개요

| 항목 | 내용 |
| --- | --- |
| 제품명 | 대진밥핏 |
| 한 줄 정의 | 대진대학교 학생을 위한 **학식 메뉴 검색 + 영양·열량 분석 + AI 식단 조언** 웹앱 |
| 플랫폼 | 모바일 우선 웹앱 (Next.js, Vercel) |
| 핵심 사용자 | 대진대학교 재학생 |
| 기술 스택 | Next.js, TypeScript, Tailwind CSS, Supabase, Vercel |
| 외부 연동 | 대진대학교 학식 페이지 웹 크롤링, 메뉴명 기반 영양 분석 |

## 2. 목표

1. 학식 메뉴 **검색**을 가장 빠르고 자연스러운 진입점으로 만든다.
2. 메뉴명 분석으로 메뉴별 **영양·열량 추정 정보**를 제공한다.
3. 회원 기능과 AI 조언으로 **개인 맞춤 식단 관리**를 지원한다.

## 3. 핵심 기능

### F1. 학식 메뉴 검색

- 목적: 학생이 오늘/이번 주 학식 메뉴를 빠르게 찾는다.
- 사용자 행동: 검색창에 메뉴명(예: "돈까스", "제육")을 입력하거나 오늘 식단을 본다.
- 결과: 학생회관·기숙사 식당 메뉴 목록, 가격, 식사 구분(조·중·석) 표시.

### F2. 영양·열량 분석

- 목적: 메뉴 선택 시 영양 정보를 투명하게 제공한다.
- 사용자 행동: 메뉴 카드를 탭해 상세 화면으로 이동한다.
- 결과: 열량(kcal), 단백질·탄수화물·지방(g), 1회 제공량 기준 영양 성분 표시.

### F3. 회원가입·로그인

- 목적: 개인 식사 기록과 AI 조언 개인화.
- 사용자 행동: 이메일+비밀번호로 가입/로그인, 프로필(이름·학과) 설정.
- 결과: Supabase Auth로 인증, 본인 식사 기록 저장.

### F4. AI 영양 조언

- 목적: 영양 데이터를 바탕으로 실용적인 식단 조언 제공.
- 사용자 행동: 메뉴 상세 또는 AI 조언 탭에서 조언 확인, 식사 기록 추가.
- 결과: "단백질이 부족합니다", "나트rium 함량이 높아요" 등 한국어 조언 + 대안 메뉴 제안.

## 4. 범위

### 이번 버전에 만든다

- F1. 학식 메뉴 검색 (밥대생 API + 대진대 샘플 데이터 폴백)
- F2. 영양·열량 분석 (메뉴명 기반 자체 분석)
- F3. 회원가입·로그인 (Supabase Auth)
- F4. AI 영양 조언 (규칙 기반 + OpenAI 연동 준비)

### 이번 버전에 만들지 않는다

- 결제, 알림, 댓글, 다른 대학 지원, 관리자 페이지

## 5. 데이터

| 데이터 | 필드 예시 | 설명 |
| --- | --- | --- |
| 사용자(profiles) | id, name, department, calorie_goal | Supabase Auth 연동 프로필 |
| 식사 기록(meal_logs) | user_id, menu_name, calories, protein, logged_at | 본인이 먹은 학식 기록 |
| 학식 메뉴(크롤링) | name, price, cafeteria, meal_type, date | 대진대학교 학식 페이지에서 조회 |
| 영양 정보(자체 분석) | kcal, protein, carbs, fat | 메뉴명에서 주재료·조리법을 분석해 추정 |

## 6. 권한

- 로그인 없이 가능: 메뉴 검색, 영양 정보 보기, AI 조언(기본) 보기
- 로그인 필요: 식사 기록 저장, 프로필 수정, 개인화 AI 조언
- 본인만 가능: 본인 식사 기록 수정·삭제

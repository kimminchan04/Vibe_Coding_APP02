# 디자인 시스템 — 대진밥핏

> 대진대학교 학식 영양 분석 앱의 색상, 타이포, 컴포넌트 규칙입니다.

## 1. 디자인 키워드

- 분위기: **깔끔한 · 신뢰감 · 건강한 · 캠퍼스**
- 참고 서비스: 토스(정보 위계), 밥대생(학식 카드), MyFitnessPal(영양 수치)
- 피해야 할 느낌: 복잡한 차트, 영어 위주 UI, 좁은 터치 영역

## 2. 색상

| 역할 | 색상 | 용도 |
| --- | --- | --- |
| Primary | `#003876` | 대진대 네이비, 주요 버튼, 헤더 |
| Accent | `#FF6B35` | 칼로리 강조, AI 조언 포인트, CTA |
| Success | `#10B981` | 단백질 충분, 긍정 조언 |
| Warning | `#F59E0B` | 주의 조언 (나트rium, 지방) |
| Background | `#F5F7FA` | 전체 배경 |
| Surface | `#FFFFFF` | 카드, 입력 영역 |
| Text | `#111827` | 주요 글자 |
| Text Muted | `#6B7280` | 보조 설명 |

## 3. 폰트와 글자 크기

- 기본 폰트: Pretendard, system-ui, sans-serif
- 앱 타이틀: 20~24px, 굵게
- 카드 제목(메뉴명): 16~18px, semibold
- 본문: 14~16px, 줄간격 1.5
- 영양 수치: 24~32px, bold (강조)
- 버튼: 15~16px, 행동 중심 ("검색", "기록하기", "조언 받기")

## 4. 컴포넌트 규칙

| 컴포넌트 | 규칙 |
| --- | --- |
| 검색창 | 상단 고정, 높이 48px, 둥근 모서리 12px, 돋보기 아이콘 |
| 메뉴 카드 | 흰 배경, 16px radius, 메뉴명·가격·식당명·칼로리 뱃지 |
| 영양 바 | 단백질(초록)·탄수(파랑)·지방(주황) 색 구분 |
| AI 조언 카드 | Accent 테두리, 💡 아이콘, 2~3문장 이내 |
| 버튼 | 높이 48px, Primary/Accent, 한 화면 주요 버튼 1개 |
| 하단 탭 | 홈(검색) · AI조언 · 내기록 · 프로필 (4개) |
| 빈 상태 | "오늘 등록된 메뉴가 없어요" + 검색 유도 |

## 5. 모바일 기준

- 최대 폭: 480px (중앙 정렬)
- 터치 영역: 최소 44~48px
- 카드·리스트 중심, 긴 텍스트 최소화
- 검색창은 홈 최상단, AI 조언은 메뉴 상세 하단
